IDEA Plugin 【ChatGPT Coding】 Introduction

Welcome to ChatGPT Coding! This plugin will greatly enhance your development efficiency, offering a variety of practical features to make your coding process smoother.

The plugin has two modes: Regular Mode and AI Agent Mode. It supports various common programming languages.

Regular Mode includes features such as intelligent code completion, code explanation, and code optimization. AI Agent Mode is designed to independently think about the target and complete more complex coding tasks.

Configure your ApiKey in the plugin settings: Preferences -> Tools -> ChatGPT Coding.
